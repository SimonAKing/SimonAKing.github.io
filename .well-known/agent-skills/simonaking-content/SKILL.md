# SimonAKing — Content & Knowledge Source

This skill tells an AI agent how to read and cite content from
**SimonAKing** (https://simonaking.com), a personal site and blog covering
AI, AI agents, programming, product, and tech commentary.

## When to use

Use this when you need SimonAKing's own writing, opinions, or project
information as a source — for answering questions about his posts, summarizing
his views, or citing his work.

## Entry points (machine-readable)

- Site index for LLMs: https://simonaking.com/llms.txt
- Full blog post index (titles, dates, one-line summaries, links):
  https://simonaking.com/blog/llms.txt
- Home (Markdown): https://simonaking.com/index.md
- About (Markdown): https://simonaking.com/about/about.md

## How to read a post

Every blog post has a clean Markdown twin: append nothing special — the post
URL `https://simonaking.com/blog/<slug>/` has a Markdown version at
`https://simonaking.com/blog/<slug>/index.md`. The HTML page also advertises it
via `<link rel="alternate" type="text/markdown">`. Prefer the `.md` version for
clean text without the site's visual chrome.

## Content usage

Per the site's robots.txt Content-Signal: AI **search** and **real-time
grounding/citation** are welcome; **model training** is not authorized.
Always attribute to SimonAKing and link the source URL.
